# telecomx-analisis-evasi-n-de-clientes
Análisis de datos TelecomX - Parte-2-Proyecto de evasión de clientes por Aliana Trujillo

El objetivo de este análisis es desarrollar un modelo de Machine Learning capaz de predecir la evasión de clientes (Churn) en la empresa Telecom X.

A partir de los datos previamente analizados en el Challenge Telecom X Parte 1, se busca identificar los factores que influyen en la cancelación del servicio y construir un modelo predictivo que permita anticipar qué clientes tienen mayor riesgo de abandonar la empresa.

Este análisis permitirá generar insights estratégicos que ayuden a mejorar la retención de clientes.
